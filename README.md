# 基于python的简易密码管理器

一个基于Python开发的本地化密码管理器，可以安全、便捷地管理各类网站账号和密码。所有数据均经过加密后存储在本地，保障安全。

## 功能特点
- 具有注册、登录、删除、修改主密码等用户功能
- 具有添加、查看、删除、修改账号密码等密码管理功能
- 内置可设置长度和复杂度的随机密码生成器
- 使用base64、Caesar、异或加密
- 网站密码根据用户主密码加密，防止其他用户查看密码
- 不直接储存主密码，使用hash值进行登录验证

## 数据结构
使用jsonl文件储存，读入时读取为字典列表。有user_info.jsonl和user_encrypted_password.jsonl两种:  
1. 使用user_info.jsonl储存所有用户账号与主密码哈希值。具体结构如下：  
    ```jsonl
    {'username':用户名1,'encrypted_main_password':主密码哈希值1}
    {'username':用户名2,'encrypted_main_password':主密码哈希值2}
    ...
    ```
2. 使用user_encrypted_password.jsonl储存一个用户的网站、账号、加密后密码。每个用户具有唯一对应的"用户名"_encrypted_password.jsonl。具体结构如下：  
    ```jsonl
    {'weburl':网址1,'username':用户名1,'password':密码1}
    {'weburl':网址2,'username':用户名2,'password':密码2}
    ...
    ```

## 交互页面
命令行菜单式交互，共有以下两级菜单:

1. 
    用户管理菜单:<br>
    ```
        0.退出系统
        1.登录用户账号
        2.注册用户新账号
        3.修改当前用户账号主密码(未登录状态下不可用)
        4.删除当前用户账号(未登录状态下不可用)
        5.退出当前用户账号
        6.生成密码   
    ```
2. 
    账号密码管理菜单：<br>
    ```
        0.返回上一级目录
        1.查看已保存密码的网站
        2.查看网站密码
        3.修改网站密码
        4.删除网站密码
        5.新增网站密码
    ```

## 交互流程:  
- 打开程序时进入第一级菜单，可以注册、登录用户账号、生成密码、退出系统
- 登录成功后，自动进入第二级菜单，可以对网站密码进行新增、删除、修改、查看
- 在登录状态下可以从第二级菜单回到第一级菜单，可以修改当前用户账号主密码、删除当前用户账号、退出当前用户账号
